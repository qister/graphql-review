# graphql-review

## Запуск проекта

### `cd application/`
### `npm i`

### `cd ../server`
### `npm i`

Установите зависимости для клиента и сервера, перейдите в папку `server`


### `npm run dev`

Запустит однновременно сервер и клиент

### `npm run server`

Запустит отдельно сервер

### `npm run client`

Запустит отдельно клиент
